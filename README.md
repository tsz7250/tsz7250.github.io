## 🎨 作品集網頁核心特色

網頁完全基於原生 Web 技術建構，在視覺張力與動態流暢度上做了深度優化：

- 📱 **流暢響應式設計 (RWD)**：動態適應手機、平板與桌機等不同螢幕尺寸，確保介面在任何裝置上都能完美滿版。
- 📑 **動態分類分頁 (Tab Navigation)**：使用者無需動態重新整理頁面，就能在「個人專案」、「期末分組報告」與「課程作業」之間即時切換。
- 🎠 **精美卡片輪播 (Peek-Carousel)**：手動打造的專屬卡片輪播機制，整合了左右導覽鍵、進度指示點與流暢的轉場滑動特效。
- 🔍 **互動式詳情彈窗 (Detail Modal)**：輕點專案卡片即可觸發覆疊視窗，同步載入該專案的架構細節與使用的技術工具。
- 💎 **Midnight 極致暗色美學**：以藍黑色調為視覺基底，襯托出精緻的色彩標籤，並透過游標懸停的微動效堆疊出科技感。

## 🛠️ 網頁核心技術棧

- **結構與佈局**：HTML5 語意化結構、CSS Flexbox 與 CSS Grid 佈局架構
- **視覺與設計**：Vanilla CSS 變數管理、Midnight 特調色系、CSS Transitions 動態轉場
- **邏輯與交互**：Vanilla JavaScript
